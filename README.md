## 📌 Description

**ControlRouter** is a powerful and professional **WiFi Access Control Tool** built for Termux and Linux. It allows you to:
- **Scan** all devices connected to your network.
- **Block** or **Unblock** specific devices via MAC Address.
- **Whitelist** trusted devices and block everything else.
- **Automatically** monitor and block new unauthorized devices.
- **Log** all activities for auditing.

> ⚠️ **Disclaimer:** This tool is intended for **educational purposes** and for use on **your own network only**. Do not use it on networks you do not own or have explicit permission to manage.

## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | 📡 **Scan Network** | Detect all active devices (IP, MAC, Name) |
| 2 | 🚫 **Block MAC** | Block a specific device via MAC Address |
| 3 | ✅ **Unblock MAC** | Unblock a previously blocked device |
| 4 | 📋 **Blacklist** | View all blocked devices |
| 5 | 📋 **Whitelist** | View all trusted devices |
| 6 | 🛡️ **Block All New** | Block every new device not in the blacklist |

| 8 | 🔄 **Auto-Block** | Continuously monitor and block intruders |
| 9 | 📰 **Activity Logs** | View real-time logs of all actions |
| 10 | ⚙️ **Settings** | Configure gateway, interface, and more |
| 11 | 📱 **Device Info** | Detailed information about connected devices |
